A complete, web‑based Library Management System built for academic institutions. It provides a new experience for your library and is suitable for Windows and Linux environments.
It handles every library operation: circulation (issue/return/renew/fines), member management (students, faculty, staff with categories), catalog (books, eBooks, serials), acquisitions (orders, vendors), research databases, financials (payments, budgets), reports, notifications (email/SMS/WhatsApp), OPAC (online public access), self‑service kiosk with mini‑RFID, computer lab booking, multi‑cloud backup, and more – all in English, Hindi, and Arabic (RTL).

This software works in Windows and Linux environments. Please refer to the Installation Guide.

All user manuals and guides are available in the `GUIDE/` folder for quick reference.

## New & Unique Features

| Mini‑RFID – Book Exit Gate | Scan a book barcode at the kiosk – automatically exits the borrower if the book is issued. Unauthorized exits trigger sound alerts & visual warnings. Simulates RFID gates without expensive tags. |
| Fingerprint Animations | Guest In/Out buttons now feature animated fingerprint icons with a gentle bounce – giving a modern, tactile feel to the self‑service kiosk. |
| Reduced Scanner Width | The barcode input field is now narrower and more focused – optimized for a clean, professional kiosk look. |
| Comprehensive Guide Files | All documentation is now available in the `GUIDE/` folder – including Installation Guide, System Settings, Mini RFID Guide, and Competitive Edge comparison. |

# Main Modules

- Admin Dashboard – Real‑time statistics, quick actions, visitor count, library hours
- Circulation – Issue, return, renew, fines, overdue management
- Members – Manage students, faculty, staff with categories, photos, documents
- Books – Full catalog with barcode, ISBN, cover images, metadata from Google/OpenLibrary
- eBooks – Digital collection with download counter, cloud storage ready

# Research

- Research Databases – Manage academic databases, access logs, featured resources

# Acquisitions & Serials

- Acquisitions – Purchase orders, vendors, budget tracking
- Serials Control – Manage subscriptions (print/online), issue check‑in, automated reminders

# Administration

- System Settings – Central control panel with 12 tabs (General, OPAC, Circulation, Notifications, Member Portal, Visitors, Hours, WhatsApp, Backup, Cloud, Institution, Report)
- Quick Actions Manager – Customize dashboard shortcuts
- Loan Rules – Patron + item category based rules (max loans, loan period, renewals, fine)
- Staff Profiles – Manage staff photos and designations displayed on login page
- Manage Notices – Create and publish library announcements
- Suggestions – Collect and manage member suggestions
- Fine Rates – Per category fine rates with grace period and max fine
- Backup & Restore – One‑click SQL backup with cloud storage (Google Drive, Dropbox, OneDrive, pCloud, MEGA, Backblaze, Wasabi, S3)
- Utilities Suite – ID cards, barcodes/labels, ISBN metadata (Google Books, OpenLibrary, ISBNdb), PDF toolbox (merge, split, watermark), shelf/spine labels
- Stock Verification – Inventory management and verification
- Lost & Damaged – Manage lost/damaged books, replacement costs
- Payments – Fine payments, receipts, transaction history
- Reservations – Book hold management with notifications
- Library Events – Event management with attendee tracking
- Member Documents – Upload and verify member documents (ID proofs, photos)
- Computer Access – PC booking, session tracking, usage policies, attendance compliance
- Data Import – Bulk import from CSV, Excel, JSON, XML, MARC
- Entry/Exit – Real‑time visitor tracking, guest mode, purpose selection, live statistics (gender, category, purpose), auto‑generated guest IDs
- Session Logs – Track user sessions and activities

# Communication

- Bulk Messaging – Email, SMS, WhatsApp with templates, placeholders (`{member_name}`, `{book_title}`, `{days_overdue}`, `{fine_amount}`), scheduling, test mode
- Chatbot – AI‑powered conversational assistant with optional OpenAI integration, persistent chat, member context (loans, fines), rate limiting

# Reports & Analytics

- Reports – Advanced reports with class/division filters: Members, Books, Circulation, Entry/Exit, Overdues, Computer Usage, Fine Payments. Export to PDF/Excel/CSV with logo, watermark, signature support.
- Advanced Analytics – Real‑time visitor count, trending books, most downloaded eBooks, pending reservations, overdue count
- Cron – Scheduled tasks (backup, overdue notifications, daily stats)

# Public Interfaces

- OPAC – Modern public catalog with 9 carousel styles (Cover Flow, Cube, Fade, Flip, Zoom, Stack, Slide, Classic), semantic search (OpenAI), member portal (holds, fines, loans, history), quote rotation
- Digital Library Hub – Central access to all digital resources
- Self‑Service Kiosk – Standalone kiosk with ID scanning, purpose selection, guest mode, live library hours, quotes, carousels (new arrivals, popular eBooks, notices), mini‑RFID book exit feature
- Member Portal – Members can view loans, fines, holds, history, update profile, reset password

# Multi‑Library

- Institutions – Multi‑branch support with separate settings and holdings

# API

- REST API – Full RESTful API for integration with external systems

# Account

- Logout – Secure logout with session termination
